# 技术文档

## API 设计

### 安全机制

#### 1. 请求签名验证

每个请求需携带以下 Header：

| Header | 说明 |
|--------|------|
| `X-App-Key` | 应用密钥，预置在客户端 |
| `X-Timestamp` | 时间戳（毫秒） |
| `X-Signature` | 请求签名 |

**签名算法**:
```
signature = SHA256(appKey + timestamp + requestBody)
```

#### 2. 时间戳校验

- 请求 timestamp 与服务器时间差超过 5 分钟，视为无效请求

#### 3. 请求限流

| 类型 | 限制 |
|------|------|
| IP 限流 | 100次/分钟 |
| API Key 限流 | 10000次/分钟 |

### 接口列表

#### 用户接口

```
POST /api/v1/bazi/calculate
     提交生日，获取八字结果

     Body: {
       name: string,        // 姓名
       birthday: string,    // 生日 "1990-01-15"
       hour: number,        // 出生小时 0-23
       gender: string      // 性别 "男" | "女"
     }

     返回: {
       userId: string,     // SHA256(设备ID+姓名+生日)
       bazi: {
         year: { stem: string, branch: string },   // 年柱
         month: { stem: string, branch: string },   // 月柱
         day: { stem: string, branch: string },     // 日柱
         hour: { stem: string, branch: string }     // 时柱
       },
       wuxing: { ... },    // 五行分析
       shishen: { ... }    // 十神
     }
```

```
GET /api/v1/bazi/calendar
     获取某月日历

     Query: {
       userId: string,
       year: number,
       month: number
     }

     返回: {
       days: [
         {
           date: string,           // "2025-06-01"
           ganzhi: string,         // 干支 "丙寅"
           wuxing: string,         // 五行 "火木"
           yi: string[],           // 宜做 ["祭祀", "开业"]
           ji: string[],           // 忌做 ["动土", "破屋"]
           star: string,           // 星宿
           luck: string            // 吉凶 "吉"
         }
       ]
     }
```

#### 管理后台接口

```
POST /api/v1/admin/login
     管理员登录

     Body: {
       username: string,
       password: string
     }

     返回: {
       token: string  // JWT token
     }
```

```
GET /api/v1/admin/users
     获取所有用户列表（需管理员权限）

     Query: {
       page: number,
       pageSize: number
     }

     返回: {
       users: [...],
       total: number
     }
```

## 数据模型

### Admin 表 (管理员)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| username | VARCHAR(50) | 用户名 |
| password | VARCHAR(255) | 密码（加密存储） |

### User 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| user_id | VARCHAR(64) | 唯一标识 SHA256(设备ID+姓名+生日) |
| name | VARCHAR(50) | 姓名 |
| birthday | DATE | 生日 |
| hour | TINYINT | 出生小时 |
| gender | ENUM | 性别 |
| bazi | JSON | 八字结果 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |
