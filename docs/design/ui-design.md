# 十神能量日历 · UI设计文档

> **版本**：V1.0
> **编制日期**：2026年5月14日
> **基于**：《十神能量日历_项目说明书.md》

---

## 一、设计规范

### 1.1 配色体系

| 用途 | 色值 | 色名 | 说明 |
|------|------|------|------|
| **主色** | `#C84A3E` | 朱砂红 | Logo、按钮、重点文字、节日/周末日期 |
| **辅色-财** | `#D4A843` | 暖金 | 财类十神标签、企业定制水印 |
| **辅色-绿** | `#5A8A6A` | 墨绿 | 保存按钮、成功状态 |
| **文字主色** | `#2C1810` | 墨褐 | 标题、正文 |
| **文字辅色** | `#5A4A3A` | 深棕 | 次级标题、十神名称 |
| **文字灰** | `#8B7355` | 浅棕 | 说明文字、农历日期 |
| **背景色** | `#F7F4EF` | 米白 | 页面底色 |
| **卡片背景** | `#FAF8F4` / `#FFF8F0` | 暖白 | 内容卡片、寄语区 |

### 1.2 字体规范

| 用途 | 字体 | 说明 |
|------|------|------|
| 中文标题 | 思源宋体 (Source Han Serif) / 方正清刻本悦宋 | 衬线体，文化感 |
| 中文正文 | 思源黑体 (Source Han Sans) / 苹方 | 无衬线，易读性 |
| 数字/英文 | SF Pro / Roboto | 现代感，与中文形成对比 |

### 1.3 UI气质

- **风格**：新中式极简
- **参考**：故宫文创、观夏品牌
- **禁止**：紫黑色「江湖算命」风格

### 1.4 间距系统

- 页面边距：16px（移动端）
- 卡片间距：12px
- 内部元素间距：8px / 12px / 16px
- 卡片圆角：12px

### 页面4：Tab导航结构

**底部Tab栏（4个）**：

| Tab | 图标 | 名称 | 选中态 | 未选中态 |
|-----|------|------|--------|----------|
| 月历 | 日历图标 | 「月历」 | 图标#C84A3E，文字#C84A3E | 图标#8B7355，文字#8B7355 |
| 今日 | 卡片图标 | 「今日」 | 图标#C84A3E，文字#C84A3E | 图标#8B7355，文字#8B7355 |
| 送礼 | 礼物图标 | 「送礼」 | 图标#C84A3E，文字#C84A3E | 图标#8B7355，文字#8B7355 |
| 我的 | 用户图标 | 「我的」 | 图标#C84A3E，文字#C84A3E | 图标#8B7355，文字#8B7355 |

**Tab内容页**：

1. **月历Tab**：月历页（页面2）
2. **今日Tab**：今日能量卡片页（与日详情页类似，但突出分享功能）
3. **送礼Tab**：送礼入口页（输入朋友生日，生成定制日历）
4. **我的Tab**：个人中心页（会员状态、订单历史、十神知识库、设置）

## 三、页面流程图

```
启动页 (Splash)
    │
    ▼ 点击「开启我的日历」
信息录入页 (Onboarding)
    │
    ▼ 点击「生成我的日历」
月历页 (Monthly)
    │
    ├──▶ 点击日期 → 日详情页 (Daily Detail)
    │                      │
    │                      ▼ 点击底部按钮
    │                    分享/保存/送礼/返回
    │
    └──▶ 底部Tab切换
            ├── 月历 → 月历页
            ├── 今日 → 今日能量卡片
            ├── 送礼 → 送礼页
            └── 我的 → 个人中心
```
